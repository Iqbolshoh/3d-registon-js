# 🕌 3D Registon — Samarqand Interaktiv Tajribasi

[![Three.js](https://img.shields.io/badge/Three.js-r128-black?style=for-the-badge&logo=three.js)](https://threejs.org/)
[![JavaScript](https://img.shields.io/badge/JavaScript-ES6-F7DF1E?style=for-the-badge&logo=javascript)](https://developer.mozilla.org/)

> **3D da Samarqandning qalbi — Registon maydoni**

## ✨ Live Demo

🔗 [View in Browser](https://your-username.github.io/3d-registon/)

## 🎯 Loyiha haqida

Bu **interaktiv 3D tajriba** bo'lib, foydalanuvchi Samarqanddagi Registon maydonidagi uchta mashhur madrasani:
- **Ulug'bek Madrasasi** (1420)
- **Sherdor Madrasasi** (1636)
- **Tilla-Qori Madrasasi** (1660)

ko'rish, aylantirish va ular haqida ma'lumot olish imkoniyatini beradi.

## 🚀 Xususiyatlar

| Xususiyat | Tavsif |
|-----------|--------|
| 🎮 **Orbit Controls** | Sichqoncha bilan aylantirish, zoom |
| 🖱️ **Click Interaction** | Madrasalarni bossangiz ma'lumot chiqadi |
| 🌟 **3D Animatsiya** | Yulduzlar, yorug'lik effektlari |
| 🎨 **Realistik Materiallar** | Metal, gumbaz, zarhal detallar |
| 📱 **Responsive** | Telefon va kompyuterda ishlaydi |
| 🌙 **Atmosfera** | Tuman, yulduzlar, soyalar |

## 🛠️ Texnologiyalar

- **Three.js** — 3D rendering
- **OrbitControls** — Kamera boshqaruvi
- **HTML5/CSS3** — UI va modal oyna
- **Vanilla JavaScript** — Interaktivlik

## 📂 Fayl tuzilmasi
```bash
3d-registon/
├── index.html # Asosiy struktura
├── style.css # Stil va modal
├── script.js # 3D logika
└── README.md # Hujjat
```


## 🚀 Ishga tushirish

### Lokal kompyuterda

```bash
# 1. Papkani klonlash
git clone https://github.com/Iqbolshoh/3d-registon.git
```

## 2. Papkaga o'tish
cd 3d-registon

## 3. Live server bilan ochish (yoki index.html ni to'g'ridan-to'g'ri)
 VS Code da "Live Server" plaginini ishlating

## Yangi madrasa qo'shish

script.js dagi madrasaData massiviga qo'shing:
```bash
{
    name: "Yangi Madrasa",
    desc: "Ma'lumot...",
    color: 0x00ff00,
    year: 2024,
    position: { x: 7, z: 2 }
}
```
